# Skill 竞技场 — 扩充 Skill 集（v2）

工作目录：E:\Projects\skill竞技场

## 目标
把现有 3 个 seed skill 扩充到 ~20+ 个，覆盖多领域、多风格，每个 skill 文件都标注来源（generated/v4-pro、collected/<URL>、imported/<原名>），保证可溯源、可审计。

## 三条平行轨

### Track 1: v4-pro 生成（deepseek-gen）
派 1 个 coder 任务：用 v4-pro 生成 8-12 个差异化的 skill，覆盖：
- 写作类（不同场景/受众/风格）：说服型、技术文档型、叙事型、学术型
- 编程类：code-reviewer、sql-expert、pythonic-coder
- 分析推理类：chain-of-thought、devils-advocate、data-analyst
- 元能力类：self-critic、step-back
- 格式约束类：json-only、markdown-table

每个 skill 必须 100-400 字，文件名 `skills/gen-<slug>.md`，文件头注释 `# Source: generated by deepseek-v4-pro, 2026-06-12`。

### Track 2: 网上搜集（web-research）
派 1 个 coder 任务：从公开来源搜集 6-10 个有代表性的 skill / system prompt：
- GitHub awesome-chatgpt-prompts / awesome-system-prompts 等仓库
- Anthropic Cookbook / OpenAI Cookbook 公开示例
- PromptHero / FlowGPT 等公开 prompt 库
- 学术论文里 cited 的 system prompts

每个文件 100-400 字，文件名 `skills/collected-<slug>.md`，文件头注释 `# Source: <URL> (collected 2026-06-12)`。允许 1-2 个标注 `# Source: synthesized from multiple references, see <URL1>, <URL2>`。

### Track 3: 用户导入（your-real-skills）
**不派任务**——目录 `import_skills/` 已就绪，用户自行往里丢文件。
本任务**附带**一个轻量扫描脚本 `scripts/import_user_skills.py`：把 `import_skills/` 下所有 .md/.txt/.yaml 复制到 `skills/imported-<原名>.md`，并加文件头注释 `# Source: imported from import_skills/<原名>, 2026-06-12`。要求幂等、可重复跑。

### Track 4: 集成验证（integration-verify）
派 1 个 coder 任务（依赖 Track 1/2/3 完成）：
1. 跑 `python scripts/import_user_skills.py`（即使 import_skills/ 为空也要跑，确认幂等）
2. 用 `arena.runner.load_skill` 对所有新 skill 文件做 syntax check + 长度校验（100-400 字）
3. 跑 `pytest tests/ -v` 确认没有回归
4. 写 `deliverable-skill-expansion.md`：列出每个新 skill 的文件名 + 来源标注 + 长度 + 一句话风格描述

## 关键约束
- 所有新 skill 严禁硬编码 API key
- 每个 skill 文件**必须有**文件头 `# Source: ...` 注释，否则视为来源不明
- 长度必须在 100-400 字之间（与现有 seed skill 保持一致）
- 不修改现有 3 个 seed skill（concise/detailed/structured-writer）和任务集
- Track 2 不联网下载二进制文件，只复制 prompt 文本（避免版权/安全问题）

## 交付物
- Track 1: 8-12 个 `skills/gen-*.md` + `deliverable-gen.md`
- Track 2: 6-10 个 `skills/collected-*.md` + `deliverable-collected.md`
- Track 3: `scripts/import_user_skills.py`（脚本本身）
- Track 4: `deliverable-skill-expansion.md`（总览）
